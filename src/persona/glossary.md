# Chinese Astrology Glossary

## BaZi (八字 / Four Pillars of Destiny)

| Term | Chinese | Description |
|------|---------|-------------|
| Four Pillars | 四柱 | Year, Month, Day, Hour pillars — each a Heavenly Stem + Earthly Branch pair |
| Heavenly Stems | 天干 | 甲乙丙丁戊己庚辛壬癸 (10 stems, cyclic) |
| Earthly Branches | 地支 | 子丑寅卯辰巳午未申酉戌亥 (12 branches, cyclic) |
| Day Master | 日主 | The Heavenly Stem of the Day Pillar — represents the self |
| Ten Gods | 十神 | Relationship of each stem to the Day Master |
| Hidden Stems | 藏干 | Stems hidden within each Earthly Branch |
| Five Elements | 五行 | Wood 木, Fire 火, Earth 土, Metal 金, Water 水 |
| True Solar Time | 真太陽時 | Longitude-corrected birth time for accurate hour pillar |

| 食神 | Shí Shén | Eating God, creativity, enjoyment |
| 傷官 | Shāng Guān | Hurting Officer, rebellion, talent |
| 偏財 | Piān Cái | Indirect Wealth, windfall, generosity |
| 正財 | Zhèng Cái | Direct Wealth, stability, diligence |

| 正官 | Zhèng Guān | Direct Officer, discipline, status |
| 偏印 | Piān Yìn | Indirect Seal, unconventional wisdom |
| 正印 | Zhèng Yìn | Direct Seal, nurturing, knowledge |

## Zi Wei Dou Shu (紫微斗數 / Purple Star Astrology)

| Term | Chinese | Description |
|------|---------|-------------|
| Life Palace | 命宮 | Core personality and destiny |
| Body Palace | 身宮 | Later-life development |
| Bureau | 局 | Five Elements Bureau (2-6), determines star positions |
| Zi Wei Star | 紫微星 | Emperor Star, leadership and authority |
| Tian Fu Star | 天府星 | Treasury Star, wealth and stability |
| SiHua | 四化 | Four Transformations: 化祿化權化科化忌 |

### Twelve Palaces (十二宮)
命宮 (Life), 兄弟宮 (Siblings), 夫妻宮 (Spouse), 子女宮 (Children), 財帛宮 (Wealth), 疾厄宮 (Health), 遷移宮 (Travel), 交友宮 (Friends), 事業宮 (Career), 田宅宮 (Property), 福德宮 (Fortune/Happiness), 父母宮 (Parents)

### SiHua Transformations (四化)
| Transform | Chinese | Energy |
|-----------|---------|--------|
| 化祿 | Huà Lù | Prosperity, smooth flow, abundance |
| 化權 | Huà Quán | Authority, control, ambition |
| 化科 | Huà Kē | Fame, reputation, academic success |
| 化忌 | Huà Jì | Obstruction, pressure, karmic lesson |

## Fortune Cycles

| Term | Chinese | Description |
|------|---------|-------------|
| Decade Luck | 大運 | 10-year fortune cycle |
| Annual Fortune | 流年 | Year-by-year fortune |
| Tai Sui | 太歲 | Annual Grand Duke — clashes bring challenges |
| Lichun | 立春 | Start of Spring (~Feb 4) — true Chinese New Year for astrology |

### Tai Sui Clashes (犯太歲)
| Type | Chinese | Severity |
|------|---------|----------|
| 值太歲 | Zhí Tài Suì | Birth year return (medium) |
| 沖太歲 | Chōng Tài Suì | Direct clash (high) |
| 刑太歲 | Xíng Tài Suì | Punishment (medium-high) |
| 破太歲 | Pò Tài Suì | Destruction (medium) |
| 害太歲 | Hài Tài Suì | Harm (medium) |
