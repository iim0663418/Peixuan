# 佩璇的智慧結晶v1

一個基於八字命理的純前端應用程式，提供用戶輸入、八字排盤、十神與五行分析、流年互動提示等功能。

## 功能特色

*   **用戶輸入介面**: 支援出生年、月、日、時、性別、時區輸入，並可切換國曆/農曆。
*   **農曆國曆轉換**: 準確的雙向轉換及閏月判斷。
*   **八字排盤**: 自動生成四柱，標註日主。
*   **十神分析**: 自動標記十神並提供分佈表。
*   **五行能量分析**: 數值化與圖表化（雷達圖、條形圖）五行分佈，提示偏旺偏枯。
*   **大運流年**: 自動計算起運年，提供流年時間軸查詢。
*   **流年互動**: 標記流年與命盤的刑沖合害等關係。
*   **PWA 支援**: 可安裝應用程式並支援離線查詢與快取。
*   **本地紀錄**: 儲存查詢紀錄，方便管理。
*   **國際化準備**: 預設繁體中文，架構支援多語系擴充。
*   **會員功能預留**: UI 已預留登入/會員區塊。

## 技術棧

*   **前端框架**: Vue 3
*   **開發工具**: Vite
*   **程式語言**: TypeScript
*   **圖表函式庫**: Chart.js
*   **路由管理**: Vue Router
*   **農曆轉換**: [lunar-javascript](https://github.com/6tail/lunar-javascript) (透過 CDN 載入)
*   **本地儲存**: IndexedDB / LocalStorage
*   **測試框架**: Vitest

## 安裝與運行

1.  **安裝依賴**:
    ```bash
    npm install
    ```
2.  **啟動開發伺服器**:
    ```bash
    npm run dev
    ```
3.  **建置專案**:
    ```bash
    npm run build
    ```
4.  **預覽建置結果**:
    ```bash
    npm run preview
    ```

## 專案結構

```
bazi-app-vue/
├── public/
│   └── js/
│       └── lunar.min.js      # 農曆函式庫 (CDN 版本)
├── src/
│   ├── assets/               # 靜態資源 (圖示等)
│   ├── components/           # Vue 元件 (排盤、輸入、圖表、時間軸等)
│   │   └── __tests__/        # 元件測試檔案
│   ├── i18n/                 # 國際化語系檔案
│   ├── router/               # Vue Router 設定
│   ├── store/                # 狀態管理 (查詢紀錄、用戶資料)
│   ├── utils/                # 工具函式 (天干地支、五行、十神計算)
│   ├── views/                # 頁面元件
│   ├── App.vue               # 主要應用程式元件
│   ├── main.ts               # 應用程式進入點
│   └── style.css             # 全域樣式
├── index.html                # HTML 入口檔案
├── package.json              # 專案依賴與腳本設定
├── vite.config.ts            # Vite 設定檔
└── tsconfig.json             # TypeScript 設定檔
```
